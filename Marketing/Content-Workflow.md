# Content Workflow

The path every piece of content follows from the mat to publish:

```
Instructor Capture → Upload → Marketing Edit → Caption → Approval → Schedule → Publish → Report
```

| Stage | Owner | What happens |
|-------|-------|--------------|
| Instructor Capture | Instructors | Capture raw video + photos during class (see Content Capture SOP) |
| Upload | Instructors | Upload all files to the designated shared drive by end of week |
| Marketing Edit | Marketing (Ivy) | Edit clips into reels using brand kits & templates |
| Caption | Marketing (Ivy) | Write the caption + CTA, tag the content pillar |
| Approval | Leadership | Review and approve before scheduling |
| Schedule | Marketing (Ivy) | Schedule to the content calendar |
| Publish | Marketing (Ivy) | Post goes live |
| Report | Marketing (Ivy) | Track performance and report back |
